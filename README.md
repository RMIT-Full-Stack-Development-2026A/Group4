# TicTacToang – Online TicTacToe Gaming Platform


**TicTacToang** is a modern, high-performance full-stack web platform that brings the classic Tic-Tac-Toe game into the digital era with single-player AI, local multiplayer, and real-time online multiplayer features.

## 🛠 Tech Stack

### Frontend
- React + Vite
- Vanilla CSS + Tailwind CSS
- Responsive Design (Mobile + Tablet + Desktop)

### Backend
- Node.js + Express.js
- MongoDB (Mongoose ODM)
- JWT (JWS) Authentication
- WebSocket (Socket.io)

### Architecture
- **N-Tier (Layer-based)** Architecture
- **Modular Monolith** (Medium + Ultimo level)
- Repository Pattern + DTOs

### Deployment
- Frontend & Backend deployed on **Vercel**

## ✨ Key Features Implemented

### ✅ Completed (All Levels)
- **Player Registration & Login** (Simplex + Medium + Ultimo)
- **Profile Management** with game history, search & filter
- **Local 2-Player Game** + **Single Player vs AI** (Easy / Medium / Hard)
- **Admin Panel** (manage players & game rooms)
- **Premium Subscription** (local wallet + simulated Stripe)
- **Real-time Online Multiplayer** with WebSocket
- **Real-time Chat** during online games
- **Move Recording & Replay System** (with algebraic notation)
- **Responsive UI** + Reusable Component Packages (Ultimo)
